# HTML from Epub

Extract html library from .epub file.

## Install

``` bash
$ pip install html_from_epub
```

## Use

As command line tool:

``` text
To-do
```

Use in code:

``` python
>>> from html_from_epub import html_from_epub
>>> html_from_epub('path_to_file.epub')
```

## Contribute
